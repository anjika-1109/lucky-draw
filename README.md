# 🎰 Lucky Draw Game

A colorful console-based Lucky Draw game built in Python. Players pick a number between 10 and 20 and get a chance to win exciting prizes!

---

## 🎮 How to Play

1. Run the program
2. View the available gifts and their lucky numbers
3. Enter a number between **10 and 20**
4. If your number matches the randomly selected lucky number — **you win a prize!**
5. Choose to play again or exit

## 🛠️ Tech Stack

- **Language:** Python 3
- **Libraries:** `colorama`, `random`, `time`, `sys`

---

## ⚙️ Installation & Setup

**1. Clone the repository**
```bash
git clone https://github.com/your-username/Lucky-Draw-Python.git
cd Lucky-Draw-Python
```

**2. Install dependencies**
```bash
pip install colorama
```

**3. Run the game**
```bash
python lucky_draw.py
```
